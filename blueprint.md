# 로또 번호 추첨기

## 개요

이 프로젝트는 사용자가 버튼을 클릭하여 로또 번호를 생성할 수 있는 간단한 웹 애플리케이션입니다. 최신 웹 기술인 웹 컴포넌트, 모던 CSS 및 ES 모듈을 사용하여 구축합니다.

## 기능

-   **번호 생성:** 사용자는 버튼을 클릭하여 1부터 45까지의 6개 고유한 랜덤 번호를 생성할 수 있습니다.
-   **애니메이션:** 생성된 번호는 애니메이션 효과와 함께 표시됩니다.
-   **반응형 디자인:** 애플리케이션은 모든 화면 크기에서 잘 보입니다.
-   **다크/라이트 모드:** 사용자는 시스템 설정이나 버튼을 통해 다크 모드와 라이트 모드를 전환할 수 있습니다.

## 디자인 및 스타일

-   **색상 팔레트:** 활기찬 그라데이션과 그림자 효과를 사용하여 깊이감과 시각적 흥미를 더합니다.
-   **타이포그래피:** 가독성 좋고 현대적인 글꼴을 사용합니다.
-   **아이콘:** 사용자 인터페이스를 향상시키기 위해 아이콘을 사용합니다.
-   **애니메이션:** 버튼과 번호 표시에 미묘한 애니메이션을 추가하여 사용자 경험을 향상시킵니다.

## 현재 진행 작업

**리팩토링: 다크/라이트 모드 버튼 디자인 및 위치 변경**

-   **`index.html`:**
    -   기존의 토글 스위치 UI를 아이콘 기반의 버튼으로 변경하여 사용자 경험을 개선합니다.
-   **`style.css`:**
    -   테마 전환 버튼의 위치를 화면 우측 상단으로 이동시킵니다.
    -   버튼에 해(라이트 모드)와 달(다크 모드) 아이콘을 추가하고, 세련된 애니메이션 효과를 적용하여 시각적인 매력을 더합니다.
-   **`main.js`:**
    -   변경된 UI 구조에 맞게 이벤트 리스너를 수정합니다.

## 구현 계획

1.  **`index.html`:**
    -   페이지 제목을 "로또 번호 추첨기"로 변경합니다.
    -   로또 번호와 생성 버튼을 담을 컨테이너를 추가합니다.
    -   생성된 각 번호를 표시하기 위한 웹 컴포넌트를 사용합니다.
2.  **`style.css`:**
    -   전체적인 레이아웃, 색상, 글꼴 등 애플리케이션의 스타일을 지정합니다.
    -   모바일 우선 접근 방식으로 반응형 디자인을 구현합니다.
    -   버튼과 번호 표시에 "빛나는" 효과와 그림자를 추가합니다.
3.  **`main.js`:**
    -   `lotto-ball` 웹 컴포넌트를 정의하여 각 번호를 표시합니다.
    -   로또 번호를 생성하는 로직을 구현합니다.
    -   버튼 클릭 시 번호를 생성하고 표시하는 이벤트 리스너를 추가합니다.
    -   번호가 나타날 때 애니메이션 효과를 추가합니다.
